# Arduino workbench controller

![](./res/image.jpg)

Arduino uno r4 dashboard for interacting with philips hue server[^server]. Built with [PlatformIO](https://platformio.org/).

- **left**: dim light.
- **center**: toggle light.
- **right**: brighten light.
- **most right**: toggle and update the current weather temperature. Connects to [weatherapi](https://www.weatherapi.com/)

[^server]: [IoT Server](https://github.com/iacobucci/iot-server)
